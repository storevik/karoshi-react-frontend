## Production image

### Prerequisites

- Docker
	- If running on Windows you need WSL2 to run Linux images.
- docker-compose

### How to run it

- Run `docker-compose up -d` in project root
- Visit http://localhost:3000 in browser
- Done

You can run `docker-compose down` to take the container down again.
